# Project Context

## Mission
Enhance the existing Compliance & Risk Control Tower into an Enterprise AI Governance, Risk, Compliance & Portfolio Platform (AI-GRCP).

## Guiding Principle
- Enhance, do NOT rewrite.
- Refactor incrementally.
- Preserve existing routes, APIs, permissions and reusable components.
