# AI-GRCP Delta Enhancement Backlog

This package contains a split, incremental backlog to enhance the existing **Enterprise AI-Powered Compliance & Risk Control Tower** without rewriting the overall design.

The objective is to add the **delta features** discussed:
- Separate **Roadmaps Monitoring** and **Projects Monitoring**
- Monthly/ad hoc roadmap snapshot engine
- Capital Markets roadmap and project dashboards
- Portfolio intelligence
- AI Hub, AI chatbot, agents, knowledge base and MCP layer
- Reporting, KPI, workflow and notification engines
- Implementation prompts usable with OpenCode + DeepSeek V4 Flash

## How to use

Use the files in this order:

1. `01_MASTER_GUIDELINES.md`
2. `02_EXECUTION_PLAN.md`
3. `03_EPIC_INDEX.md`
4. Then execute each epic file one by one.

Each epic contains:
- Business objective
- Scope
- User stories
- Technical tasks
- Acceptance criteria
- Suggested OpenCode prompt
- Risks
- Deliverables

## Critical rule

Do not ask the coding model to implement everything at once.

Use one epic at a time.
